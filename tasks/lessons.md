## 2024-02-24: Initial Setup

### Правила кода
- **z-index**: Использовать Tailwind scale (z-10, z-20, z-50), не magic numbers
- **Portals**: Для модалок/тултипов использовать React Portal если нужен escape из overflow
- **TypeScript**: Никаких `any`, всегда типизировать
